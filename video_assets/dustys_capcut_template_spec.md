# Dusty's CapCut Template Spec

Use this to create a reusable CapCut project template for Dusty's short videos.

---

## Project Setup

- Canvas: 9:16
- Resolution: 1080x1920
- Frame rate: 30fps
- Duration presets: 15 sec and 30 sec
- Safe zones: leave top/bottom padding for platform UI

---

## Track Layout

| Track | Content |
|---|---|
| V1 | main footage |
| V2 | B-roll / close-up inserts |
| V3 | text overlays |
| V4 | end card / logo badge |
| A1 | original audio / voice |
| A2 | BGM |
| A3 | sound effects |

---

## Text Styles

### Hook text

- All caps
- White fill
- Orange keyword
- Black shadow/stroke
- Large enough for mobile
- Duration: 1.5–2.5 seconds

Examples:

```text
DON'T RIDE THIS
USED BIKE RED FLAG
TEXT A PHOTO
BEFORE THE CANYON
```

### Explainer captions

- Sentence case
- White fill
- Subtle black shadow
- Lower third
- 1–2 lines max

### CTA card

```text
Text Dusty a photo
714-787-9453
Orange, CA
No phantom repairs. Quote before we wrench.
```

---

## Color Preset

Name: `Dusty Garage Clean`

- Exposure: adjust per clip
- Contrast: +8 to +15
- Saturation: +4 to +8
- Temperature: slightly warm if shop light is cool
- Shadows: lift only enough to show part detail
- Highlights: reduce if metal glare is blown out

---

## Sound Presets

### Voice cleanup

- AI noise reduction: light/medium
- High-pass filter: 80–120 Hz if available
- Voice enhancement: light
- BGM ducked under voice

### SFX

Use sparingly:

- soft click for text pop,
- low whoosh for transition only if needed,
- no cartoon effects for serious safety content.

---

## Template Variants

### Variant A — Repair Red Flag

- Hook overlay at 0s.
- Problem close-up first.
- 2 explainer captions.
- CTA end card.

### Variant B — Used Bike Inspection

- Checklist-style overlays.
- 3–5 quick cuts.
- CTA: send listing/photos.

### Variant C — Ride Ready

- Fast checklist rhythm.
- Each item gets one shot.
- CTA: text before the ride.

### Variant D — No-Upsell Proof

- Problem part.
- What can wait.
- Quote-before-wrenching line.
- CTA end card.
