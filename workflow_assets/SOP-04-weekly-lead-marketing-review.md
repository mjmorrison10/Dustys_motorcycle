# SOP-04 — Weekly Lead & Marketing Review

## Goal

Use 20 minutes per week to learn which channels create qualified work and which workflows need fixing.

## Cadence

Every Friday or Monday morning.

## Inputs

- lead log,
- completed jobs,
- review tracker,
- website/GA4 events if available,
- GBP insights if available,
- social posts/leads.

## Questions

1. How many leads came in?
2. How many were qualified?
3. Which sources created qualified leads?
4. Which leads lacked bike info/photo/source?
5. Which jobs were completed?
6. Which customers are review-ready?
7. Which experiment should continue, pause, or change?

## Outputs

- one operational fix,
- one content/social action,
- one review/referral action,
- one measurement note.

## Done When

- lead log updated,
- review asks assigned,
- next week experiment chosen,
- blockers noted.
