# SOP-03 — Post-Service Review & Referral Workflow

## Goal

Turn completed successful jobs into reviews, referrals, and repeat trust.

## Trigger

Job completed and bike picked up.

## Steps

1. Log job complete.
2. Mark customer sentiment.
3. Check if unresolved issue exists.
4. Send check-in 2–7 days later.
5. If positive, ask for review.
6. If review/positive repeat customer, ask for referral.
7. Set 90–180 day reminder if appropriate.

## Check-In Message

```text
Hey [Name], just checking in — how’s the [bike] feeling after pickup?
```

## Review Ask

```text
Glad to hear it. If Dusty treated you fairly, would you mind leaving a quick Google review? It helps local riders find an honest shop.
[Google Review Link]
```

## Referral Ask

```text
If you know another rider who needs a straight-answer mechanic in Orange, send them Dusty’s number: 714-787-9453.
```

## Do Not Ask For Review If

- unresolved issue,
- customer upset,
- warranty concern open,
- job outcome uncertain.
