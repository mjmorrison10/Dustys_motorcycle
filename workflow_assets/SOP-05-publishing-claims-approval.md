# SOP-05 — Publishing & Claims Approval

## Goal

Prevent inaccurate public claims about prices, hours, warranty, address, discounts, customer proof, and safety.

## Requires Dusty Approval

- prices,
- warranty terms,
- hours,
- address,
- discounts/referral incentives,
- financing/payment plans,
- exact response-time promises,
- customer names/faces/plates/bikes,
- before/after claims,
- safety claims.

## Safe Without Special Approval

- general maintenance education,
- checklists,
- “quote before we wrench” process language,
- “text year/make/model + issue + photo/video,”
- no-BS / no phantom repairs positioning already used on site.

## Publishing Checklist

- [ ] Claim is accurate.
- [ ] Owner approval captured if needed.
- [ ] No private customer info exposed.
- [ ] No fake AI proof.
- [ ] No exact price unless approved.
- [ ] No unsafe remote diagnosis.
- [ ] NAP/hours consistent.

## Done When

Public content is accurate, approved where needed, and logged if it changes core business claims.
