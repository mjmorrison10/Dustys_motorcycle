# Dusty's Growth Dashboard Spec

Use this as the dashboard spec for GA4, Looker Studio, Airtable, Google Sheets, or a manual weekly review.

## North Star Metric

Qualified leads per month with:

- year/make/model,
- issue/service need,
- photo/video or clear description,
- source captured.

## Weekly Metrics

| Metric | Source | Target |
|---|---|---:|
| Website sessions | GA4 | baseline + growth |
| SMS CTA clicks | GA4/GTM | increasing |
| Form submissions | form/CRM | increasing after endpoint live |
| Guide clicks | GA4/GTM | 5–10% engaged visitors |
| Text-builder interactions | GA4/GTM | 10%+ section visitors |
| Referral share clicks | GA4/GTM | 10+/month after launch |
| Calls | GBP/site | baseline + growth |
| Directions clicks | GBP/site | baseline + growth |
| Leads by source | CRM/manual | 80% tagged |
| Reviews requested | manual/CRM | 80% of completed good jobs |
| Reviews received | Google/Yelp | 8–15/month target |

## Weekly Review Questions

1. Which channel produced the best qualified leads?
2. Which CTA drove the most contact intent?
3. Which leads lacked bike info/photo and why?
4. Which experiment should continue next week?
5. Which experiment should be killed or changed?
